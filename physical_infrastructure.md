## Data Center:
- The physical facility that host these servers are called Data centers.
- Physical facility (VM, web services, sql)
- Hosting for group of networked servers.
- Own power, cooling & networking infrastructure.

### How to create a resource and add specific region:
- Create a resource/ Click storage account
- add
- Location selection:
  - Go to the website called 'Azure speed test'
  - You will see the avg latency.
  - Pick up first 2 region's names.
  - Now go to 'azure products by region' / 'products available by region'
  - Select those previous regions.
  - Click 'select all products'
  - Now you can see which region is better , what services are available to which region.

## Availability Zone:
- Regional features
- Grouping of physically separate facilities.
- Designed to protect from data center failures.


<img width="400" alt="image" src="https://github.com/pilipi-puu-puu/Microsoft_Azure/assets/87390353/13f75213-d273-49a1-be73-4a6a6f79ef89">


- Categories:
  - Zonal services(VM, Disks etc.)
  - Zone redundant services (SQL, storage etc.)
 
  
- Key notes:
  - Not all regions are supported.
  - supported region has three or more zones
  - A zone is one or more data centers.
